Browser

A base class for all other browser models to extend.

Instance Methods

this.addCapability(key, value)

This is a simple method to define capabilities that'll be passed to the WebDriver builder when you call getSeleniumDriverBuilder() or getSeleniumDriver().

Parameters
key string

The capability key.
value string

The Value of the capability.

this.getDriverModule() → String

Returns String

Gets the name of the driver module to use with this browser. Null if there is no driver module

this.getId() → String

Get the ID of the browser.

Returns String

ID of this browser.

this.getPrettyName() → String

A user friendly name for the browser

Returns String

A user friendly name for the browser

this.getSeleniumOptions() → SeleniumOptions

The selenium options passed to webdriver's Builder method. This will have the executable path set for the browser so you should manipulate these options rather than create entirely new objects.

Returns SeleniumOptions

An instance of either selenium-webdriver/firefox or selenium-webdriver/chrome

this.setSeleniumOptions(options)

If changes are made to the selenium options, call this method to set them before calling getSeleniumDriver.

Parameters
options SeleniumOptions

An instance of selenium-webdriver/firefox or selenium-webdriver/chrome
